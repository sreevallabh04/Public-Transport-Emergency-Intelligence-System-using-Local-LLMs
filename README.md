# Smart Bus Passenger UI (Flutter - UI Only)

Academic demo UI for the project:
**“Smart Bus Accident & Passenger Health Monitoring System.”**

## What this is
- **UI screens only** (static/dummy data)
- **No backend**, no BLE, no sensors, no business logic
- Seat-based passenger identification (e.g., **Seat B12**, **Passenger P-012**)

## Screens included
- Passenger Setup / Seat Information (first screen)
- Dashboard
- Passenger Health Details
- Emergency Alert

## Run (after creating a Flutter environment)
```bash
flutter pub get
flutter run
```
